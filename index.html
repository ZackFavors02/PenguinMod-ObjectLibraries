PenguinMod Object Libraries

This site contains all PenguinMod sounds, costumes or other items that are added to the library.

To add an item to one of these libraries, feel free to contribute to the GitHub repository.


Note: All new sounds & costumes MUST be Public Domain (creative commons 0 or other public licenses) or MIT licensed. There are no exceptions.

If you make the content yourself and upload it here, note that your content can be used, sold and modified without your permission required or credits required.


Please check your sources to confirm the content is allowed on PenguinMod. For example, PenguinMod gets sounds from freesound.org under the Creative Commons 0 License.
